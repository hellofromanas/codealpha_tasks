# 📊 India Unemployment Analysis

## 📝 Project Overview
This project involves a comprehensive data analysis of the unemployment rate in India, with a specific focus on the sudden spike caused by the COVID-19 lockdowns in 2020. The analysis explores regional trends and visualizes the stark contrast in employment metrics before and during the pandemic.

This project is part of my tasks for CodeAlpha.

## 🗂️ Dataset
The analysis utilizes two primary datasets:
1. **Unemployment in India.csv:** General employment data across different states.
2. **Unemployment_Rate_upto_11_2020.csv:** Time-series data capturing the peak of the COVID-19 economic impact, including geographic coordinates for mapping.

## 🛠️ Technologies Used
* **Python** (Data manipulation and analysis)
* **Pandas** (Data cleaning, handling missing values, and time-series conversion)
* **Matplotlib & Seaborn** (Data visualization)
* **Jupyter Notebook / Google Colab** (Interactive execution environment)

## 📈 Key Visualizations & Insights
* **State-by-State Comparison:** A bar chart analysis revealing which regions historically face the highest average unemployment.
* **COVID-19 Impact Trend:** A time-series line graph demonstrating the massive, unprecedented spike in the national unemployment rate during the strict lockdown months of April and May 2020.

## 🚀 How to Run
1. Clone this repository to your local machine.
2. Ensure you have Python and the required libraries installed (`pandas`, `matplotlib`, `seaborn`).
3. Open the `.ipynb` notebook file using Jupyter or upload it to Google Colab.
4. Run the cells sequentially to view the data cleaning process and generated visualizations.
